## General info
A simple telegram bot that can recieve and store messages and user information, and can reply to messages.

## Requirements
* Examples in this repository use .NET Core 3.1 and .NET 6.
* [Telegram.Bot](https://github.com/TelegramBots/Telegram.Bot)

## Setup
To run this project,
1. create a bot for yourself by going to [BotFather](https://t.me/BotFather)
2. copy-paste your bot token in the mentioned section in the project and hit start
3. Now your bot is ready; go and chat with your bot and your messages will be displayed in the mentioned section.

## Configuration
You can add more functionalities for your bot. This project is just a simple template.

## Details
* You can answer to each user by selecting their message from the chart and writing your message in the answer section.
* You also can upload any file by hitting the `...` and adding your file and then hitting the upload button.
